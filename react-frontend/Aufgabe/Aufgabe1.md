# Listendarstellung

Erzeuge eine neue Komponente List, die Movie-Objekte darstellt.

Sorge dafür dass eine Meldung angzeigt wird, wenn keine Objekte existieren (das Array leer ist).

## Tipp:

- nutze Array.map für die Schleife

## STruktur:

```ts
type Movie = {
  id: string;
  title: string;
  year: number;
};
``;
```
